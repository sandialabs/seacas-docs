111 // Description:
112 // Initializes ParaView Catalyst to perform in-situ co-processing
113 // with the Python file catalyst_python_filename. This method can
114 // be called multiple times with different co-processing Python scripts.
115 // If initialization fails, co-processing will not occur in any other
116 // methods on this class.
117 // Additional arguments:
118 // UnderscoreVectors - joined vector variable names end in an underscore.
119 // ApplyDisplacements - a nodal variable named DISPL or displ is applied to
120 // the mesh node coordinates each time-step.
121 // restart_tag - if not empty, contains the current restart iteration string, ie s0001
122 // enable_logging - turn on logging in the adapter. Default is off.
123 // debug_level - enable catalyst debug output 0, 1, 2. Default is 0.
124 // results_output_filename - filename associated with the Ioss results output block.
125 // catalyst_output_directory - name of the output directory for storing Catalyst output.
126 // Default is CatalystOutput.
127 // catalyst_sierra_data - string data vector for development and debugging.
128 virtual void CreateNewPipeline(
