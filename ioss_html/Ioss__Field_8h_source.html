50 /* \brief Categorizes the type of information held in the field.
51 */
52 enum RoleType {
53 INTERNAL,
54 MESH, /**< A field which is used to define the basic geometry
55 or topology of the model and is not normally transient
56 in nature. Examples would be element connectivity or
57 nodal coordinates. */
58 ATTRIBUTE, /**< A field which is used to define an attribute on an
59 EntityBlock derived class. Examples would be thickness
60 of the elements in a shell element block or the radius
61 of particles in a particle element block. */
62 MAP,
63 COMMUNICATION,
64 MESH_REDUCTION, /**< A field which summarizes some non-transient data
65 about an entity (\sa REDUCTION). This could be an
66 offset applied to an element block, or the units
67 system of a model or the name of the solid model
68 which this entity is modelling... */
69 INFORMATION = MESH_REDUCTION,
70 REDUCTION, /**< A field which typically summarizes some transient data
71 about an entity. The size of this field is typically not
72 proportional to the number of entities in a GroupingEntity.
73 An example would be average displacement over a group of
74 nodes or the kinetic energy of a model. This data is also
75 transient. */
76 TRANSIENT /**< A field which is typically calculated at multiple steps
77 or times in an analysis. These are typically "results"
78 data. Examples would be nodal displacement or element
79 stress. */
80 };
