48 /* \brief Categorizes the type of information held in the field.
49 */
50 enum RoleType {
51 INTERNAL,
52 MESH, /**< A field which is used to define the basic geometry
53 or topology of the model and is not normally transient
54 in nature. Examples would be element connectivity or
55 nodal coordinates. */
56 ATTRIBUTE, /**< A field which is used to define an attribute on an
57 EntityBlock derived class. Examples would be thickness
58 of the elements in a shell element block or the radius
59 of particles in a particle element block. */
60 COMMUNICATION,
61 MESH_REDUCTION, /**< A field which summarizes some non-transient data
62 about an entity (\sa REDUCTION). This could be an
63 offset applied to an element block, or the units
64 system of a model or the name of the solid model
65 which this entity is modelling... */
66 INFORMATION = MESH_REDUCTION,
67 REDUCTION, /**< A field which typically summarizes some transient data
68 about an entity. The size of this field is typically not
69 proportional to the number of entities in a GroupingEntity.
70 An example would be average displacement over a group of
71 nodes or the kinetic energy of a model. This data is also
72 transient. */
73 TRANSIENT /**< A field which is typically calculated at multiple steps
74 or times in an analysis. These are typically "results"
75 data. Examples would be nodal displacement or element
76 stress. */
77 };
